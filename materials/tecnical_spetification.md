3. Описание проекта:
    Небольшой планировщик задач с визуализацией прогресса. Сайт позволяет пользователям создавать списки задач, 
устанавливать дедлайны, отмечать прогресс и видеть статистику выполненных дел в виде графиков.
4. 1) Понадобится примерно 4 страницы. Страница регистрации, страница добавления нового списка задач или просто задачи,
статистики и что-то типо календаря, где показываются дедлайны и такая информация.
   2) Из bootstrap будут использованы: Sidebars, buttons, dashboard, sign-in, calendar etc.
   3) Понадобятся шаблоны страниц для регистрации, может быть для вывода статистики.
   4) Почта, может быть телеграм, чтобы телеграм-бот отправлял уведомления о приближающемся дедлайне. 
   5) Аватарка, можно еще файл csv для списка задач с датами.
   6) --
   7) Задачи, проекты, в которые входят задачи. В базе будет хранится задача, дата в которую ее надо выполнить.
